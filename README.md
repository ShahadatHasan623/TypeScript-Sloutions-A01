1.What is TypeScript?
Ans: TypeScript is a strongly typed programming language built on top of JavaScript.
It adds static typing, interfaces, classes, generics, and many modern features that help developers write cleaner and safer code.

2.Why Use TypeScript?

<ul>
<ol>Better code quality</ol>
<ol>Detect errors early</ol>
<ol>Improved IDE support</ol>
<ol>Easier debugging</ol>
<ol>Scalable for large projects</ol>
